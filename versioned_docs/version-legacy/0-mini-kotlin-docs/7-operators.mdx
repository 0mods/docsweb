# Операторы и Математика в `Kotlin`

---

<br />

:::tip Что такое операторы?
**Операторы** — специальные символы или ключевые слова, которые выполняют операции над данными.
В Kotlin они делятся на арифметические, логические, сравнения и другие.
:::

<br />

---

<br />

## Арифметические операторы

<div align="left">

> Базовые операторы для работы с числами:
> - `+` — сложение:
>   ```kts title="Пример"
>   val sum = 5 + 3  // 8
>   ```
> - `-` — вычитание:
>   ```kts title="Пример"
>   val diff = 10 - 4  // 6
>   ```
> - `*` — умножение:
>   ```kts title="Пример"
>   val product = 2 * 6  // 12
>   ```
> - `/` — деление (целочисленное для `Int`, дробное для `Double`):
>   ```kts title="Пример"
>   val div1 = 10 / 3   // 3 (Int)
>   val div2 = 10.0 / 3 // 3.333... (Double)
>   ```
> - `%` — остаток от деления:
>   ```kts title="Пример"
>   val remainder = 10 % 3  // 1
>   ```

</div>

## Инкремент и Декремент

### Разница между `i++` и `++i`

<div align="left">

```kts title="Пример"
var a = 5
val postIncrement = a++  // postIncrement = 5, a = 6

var b = 5
val preIncrement = ++b   // preIncrement = 6, b = 6
```
> Пояснение:
> - **Постфиксный инкремент (`i++`)**:
>   - Возвращает текущее значение переменной.
>   - Увеличивает значение на 1 **после** возврата.
> - **Префиксный инкремент (`++i`)**:
>   - Увеличивает значение на 1 **перед** возвратом.
>   - Возвращает новое значение.

</div>

## Ограничения

### Допустим у вас есть число, но вы хотите его ограничить в каком-то диапазоне

<div align="left">

```kts title="Пример"
var a = 5
val least = a.coerseAtLeast(3)  // Если `a` меньше 3, то возвращает 3, иначе `a`, т.е. ограничивает результат, не меньше 3.
val most = a.coerseAtMost(30) // Если `a` больше 30, то возвращает 30, иначе `a`, т.е. ограничивает результат, не больше 30.
val result = a.coerseIn(0, 10) // Ограничивает `a` в диапазоне от 0 до 10.
```

</div>

## Математические функции

Kotlin предоставляет расширенный набор функций через пакет `kotlin.math`.

<div align="left">

> Основные функции:
> - `sqrt(x)` — квадратный корень:
>   ```kts title="Пример"
>   val root = sqrt(25.0)  // 5.0
>   ```
> - `pow(n)` — возведение в степень:
>   ```kts title="Пример"
>   val power = 2.0.pow(3)  // 8.0
>   ```
> - `abs(x)` — модуль числа:
>   ```kts title="Пример"
>   val absolute = abs(-7)  // 7
>   ```
> - `max(a, b)` / `min(a, b)` — максимум/минимум:
>   ```kts title="Пример"
>   val maximum = max(10, 20)  // 20
>   val minimum = min(5, -3)   // -3
>   ```
> - `round(x)` — округление до ближайшего целого:
>   ```kts title="Пример"
>   val rounded = round(3.6)  // 4.0
>   ```

</div>

## Примеры использования

<div align="left">

### Комбинирование операторов
```kts title="Пример"
val result = (10 + 3) * 2 - 5  // (13 * 2) - 5 = 21
```

### Инкременты в цикле
```kts title="Пример"
var counter = 0
repeat(3) {
    println(++counter)  // 1, 2, 3
}
```

### Математика в действии
```kts title="Пример"
fun calculateHypotenuse(a: Double, b: Double): Double {
    return sqrt(a.pow(2) + b.pow(2))
}

println(calculateHypotenuse(3.0, 4.0))  // 5.0
```
