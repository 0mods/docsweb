<br />

## Возможности `HollowCore`

<br />

### Поддержка нескольких модлоадеров.

<div align="left" id="hc-box">
- [x] Универсальная система регистрации игрового контента с помощью `Kotlin` делегатов.
- [x] Универсальная система пакетов с автоматической сериализацией и регистрацией пакета.
- [x] Универсальная система хранилища NBT данных для блоков/сущностей/измерений.
- [x] Универсальная система событий, а так же поддержка корутин.
</div>

<br />

<br />

### Упрощение разработки

<div align="left" id="hc-box">
- [x] Автоматическая генерация шаблонных моделей для блоков/предметов через встроенный ресурспак.
- [x] Сериализация любых объектов в NBT при помощи `KotlinX Serialization`.
- [x] Система `toml` конфига основанная на `KotlinX Serialization`.
- [x] Автоматическая генерация шаблонного `sounds.json`.
</div>

<br />

<br />

### Графика

<div align="left" id="hc-box">
- [x] Библиотека UI, основанная на ImGui, отрисовка базовых виджетов/предметов/блоков/контейнеров/слотов и т.д.
- [x] Многопоточный загрузчик `glTF` моделей с поддержкой:
  - Скелетной анимации,
  - Скиннинга,
  - Morph target'ы,
  - `PBR материалы для `Iris/Oculus`.
- [x] Поддержка эффектов из `Effekseer`.
- [x] Поддержка типов текстур: `apng`, `gif`, `jpeg`, `tiff` и т.д.
- [x] Поддежка типов аудио: `ogg`, `wav` и `mp3`.
</div>

## Установка `HollowCore`

<br />
<br />

### Для игры
:::note Для игры
Скачать для игры можно [**`здесь`**](../../1-hollowengine/98-downloads-mods.mdx#HollowCore)
:::

<br />

### Для разработки
:::info Для разработки
Добавьте в ваш проект репозиторий `0mods.team`:
<div align="left">
```kts
repositories {
  maven("https://maven.0mods.team/")
}
```
</div>

После, установите сам `HollowCore`:
<div align="left">
```kts
dependencies {
  implementation("ru.hollowhorizon:HollowCore-forge-1.19.2:2.0.4:dev")
}
```
Вместо `forge` - вы можеет указать любой из загрузчиков (`fabric`, `neoforge`).
А в качестве версии - поддерживаются `1.19.2`, `1.20.1` и `1.21`.
</div>
:::
