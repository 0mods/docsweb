# Поддержка аудио

---

:::note
`HollowCore` позволяет использовать не только `ogg` формат аудио, но так же `mp3` и `wav`.
К тому же - без необходимости регистрации звука.
:::

<br />

## Чтение аудио файлов
:::info Чтение аудио файлов
<div id="dont-copy">
<Tabs>
  <TabItem value="ogg-audio" label="'ogg' формат" attributes={{className: tab.def}}>

```kts
val audio = OggFormat.read(stream) // stream - всё как в Java: InputStream, поток ввода данных
```

  </TabItem>
  <TabItem value="mp3-audio" label="'mp3' формат" attributes={{className: tab.def}}>

```

  </TabItem>
  <TabItem value="wav-audio" label="'wav' формат" attributes={{className: tab.def}}>

```

  </TabItem>
</Tabs>

## Проигрывание звука

<br />

### Создание плеера
<div id="dont-copy">
```kts title="Пример"
val buffer = AudioPlayer(audio) // Создание буфера для звука
val player = AudioPlayer(buffer) // Создание проигрывателя звука
```
</div>

<br />
