# Типы данных в `Kotlin`

---

<br />

Есть несколько основных типов данных которыми пользуются чаще всего. Про них и будет речь.

<br />

---

<br />

## Числовые

<div align="left">

> - `Int` - целые числа.
>   - Допустимый диапазон: от `-2147483648` (`MIN_VALUE`) до `2147483647` (`MAX_VALUE`)
>   - Пишется как:
>     ```kts title="Пример"
>     val i = 10
>     ```
> ---
> - `Double` - Десятичные (не целые) числа.
>   - Допустимый диапазон: от `4.9E-324` (`Double.MIN_VALUE`) до `1.7976931348623157E308` (`Double.MAX_VALUE`)
>   - Пишется как:
>     ```kts title="Пример"
>     val d = 19.252
>     ```
> ---
> - `Float` - Плавающая точка.
>   - Допустимый диапазон: от `1.4E-45F` (`Float.MIN_VALUE`) до `3.4028235E38F` (`Float.MIN_VALUE`)
>   - Пишется как:
>     ```kts title="Пример"
>     val f = 82f
>     // или
>     val f = 72.53f
>     ```

</div>

<br />

---

<br />

## Строка/Текст

<div align="left">

> - `String` - Строка или Текст.
>   - Пишется как:
>     ```kts title="Пример"
>     val s = "Hello World"
>     ```

</div>

## Списки

<div align="left">

> - `List` - Динамическй список. Можно изменить его размер в любой момент.
>   - Пишется как:
>     ```kts title="Пример"
>     val listInt = listOf(0, 1, 6, 92, 333, 2)
>     val listString = listOf("Egor", "Jora", "Kirill")
>     
>     /* Расширить список можно так */
>     val dynList = listOf(0 ,1 ,4, 5)
>     dynListInt += 10 // Добавляем в список новое число 10
>     ```
> ---
> - `Array` - Статическйи список. Размер списка устанавливается изначально и остатся таким всегда.
>   - Пишется как:
>     ```kts
>     val arrayInt: Array<Int> = arrayOf(0, 2, 6, 81. 992, 3)
>     val arrayString: Array<String> = arrayOf("Kate", "Yula", "Sasha")
>     ```
> ---
> - `Map` - Статический индексированный список.
>   - Пишется как:
>     ```kts title="Пример"
>     val mapInt = mapOf(
>     /* Индекс to Значение */
>       0 to "Jora",
>       9 to "Egor",
>       2 to "Kolya",
>       61 to "Misha"
>     )
>     val card_bank = mapOf(
>     /* Индекс to Значение */
>       "balance" to 125_253,
>       "id" to 9928_8255_9252_1299,
>       "safe-code" to 912
>     )
>     ```
> ---
> - `MutableMap` - Динамический индексированный список.
>   - Пишется как:
>     ```kts title="Пример"
>     val camp = mutableMapOf(
>     /* Индекс to Значение */
>       "kate" to "kate",
>       "egor" to "Egor"
>     )
>     
>     /* Расширить список можно так */
>     val dynMap = mutableMapOf(
>       0 to "zero",
>       1 to "one",
>       2 to "two"
>     )
>     dynMap[3] = "three" // Добавляем в список индекс 3 со значением three
>     ```

</div>

## Логический

<div align="left">

> - `Boolean` - Логический тип данных, хранящий в себе только 2 данных. А именно:
>   - `false` - т.е. как 0,
>   - `true` - т.е. как 1.
>   - Пишется как:
>     ```kts title="Пример"
>     val boolFalse = false
>     val boolTrue = true
>     ```

</div>
