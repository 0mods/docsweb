# Kotlin Documentation (MINI-ATX Edition)

---

<br />

Данная документация по `Kotlin` предназначена для вас - **лентяев**!

Даже не думайте что тут будет полная документация по `Kotlin`, иш чё захотели.

В основном тут будут затронуты только **базовые** вещи и то что относится к **базовому**
скриптинку в `HollowEngine` (удивительно, правда? Вы неужели думали что доки должны вкл всё что
есть вообще кроме `HollowEngine`? Мне как и другим делать буд-то нечего как расписывать методы
майна и `Kotlin`. Хотите сделать что-то крутое на `HollowEngine`? - Берёте, изчаете язык
`Kotlin`, после изучаете как работает сам майн на `Forge`/`Fabric`/`NeoForge` и калякает всё что
вам вздумается).
