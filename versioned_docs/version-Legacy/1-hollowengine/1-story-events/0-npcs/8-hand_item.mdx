## Отображение предмета в руке

<br />

:::info Не отображается предмет
Есть после выдачи предмета в руки НИПу предмет не отображается в руках, то жмите на кнопку ниже.

<br />

<ToggleText buttonText="Предмет не отображается [Инструкция]">

Если вы используете встроенную модель и предмет не отображается в руках - тогда это странно,
так не должно быть. Сообщите `_BENDY659_` или `HollowHorizon` об этой проблеме.

<br />

Если же вы используете свою модель, то тогда следуйте инструкциям ниже!

<div align="left">

1. Откройте свою модель (где открывать - зависит от того где вы делали модель).
2. Выберите левую руку и создайте там группу/папку/кость с именем `LeftHandItem` и переместите
её ниже, чтоб она была как бы в руке.
3. Выберите правую руку и создайте там группу/папку/кость с именем `RightHandItem` и переместите
её ниже, чтоб она была как бы в руке.
4. Всё готово. Проверяйте. Если у вас всё равно не отображаются предметы в руке - попробуйте
ещё раз пройтись по пунктам и перепроверить всё.

</div>

</ToggleText>

:::

## Действие с предметами

<br />

::::note Действие с предметами

<Tabs>
<TabItem value="give-item" label="Выдать" attributes={{ className: tab.def }}>
:::info Выдать предмет в руки НИПа

```kts
/* Выдать предмет в Левую руку */
<npcID> giveLeftHand { item("<itemID>") }

/* Выдать предмет в Правую руку */
<npcID> giveRightHand { item("<itemID>") }
```
```kts title="Пример"
npc giveLeftHand { item("me:chiken") }
npc giveRightHand { item("me:cheese") }
```

:::
</TabItem>
<TabItem value="remove-item" label="Убрать" attributes={{ className: tab.def }}>
:::info Убрать предмет из рук НИПа

```kts
/* Убрать предмет в Левую руку */
<npcID> giveLeftHand { null }

/* Убрать предмет в Правую руку */
<npcID> giveRightHand { null }
```
```kts title="Пример"
npc giveLeftHand { null }
npc giveRightHand { null }
```

:::
</TabItem>
<TabItem value="drop-item" label="Выбросить" attributes={{ className: tab.def }}>
:::info Выбросить предмет от лица НИПа

```kts
<npcID> dropItem { item("<itemID>") }
```
```kts title="Пример"
npc dropItem { item("me:pineapple") }
```

:::
</TabItem>
</Tabs>
