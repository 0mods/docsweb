## Меню

:::info Меню
Есть специальное меню, где будет список всего что вы добавили в менеджер ивентов. Открывается оно
на клавишу `Ё` (по умолчанию).

:::

## Управление ивентами

:::info Управление ивентами

<div align="left" id="dont-copy">

```kts
<player>.progressManager.addMessage { "<text>" } // Добавляет сообщение в менеджер ивентов
<player>.progressManager.removeMessage { "<text>" } // Удаляет сообщение из менеджера ивентов
<player>.progressManager.clear() // Очищает менеджер ивентов
```

---

```kts title="Пример"
players.progressManager.addMessage { "Hello World!" }
players.progressManager.addMessage { "[Новый друг] Принеси к себе в хату нового друга." }
```
