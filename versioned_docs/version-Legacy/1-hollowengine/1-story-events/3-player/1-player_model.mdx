# Модель игрока

Менять модель игрока так же

![no-title](@site/static/img/doc/titles/NO_TITLE.webp)

как и у НИПов

---

<br />

<div id="dont-copy">

## Установить модель

<br />

:::info Установить модель
Установка модели происходит так же, как и у НИПов, в методе `configure`
<div align="left">

```kts
<player>.configure {
  model = "<path/to/model>" // Путь к модели
  
  // Если голова поворачивается не правильно
  switchHeadRot = true
}
```

---

```kts title="Пример"
me.configure {
  model = "my_mod:models/player/best_model.gltf"
}
```

</div>
:::

<br />

---

<br />

## Сбросить модель

<br />

:::info сбросить модель
Если же нужно вернуть стандартную модель игрока, то за место пути, укажите значение `%NO_MODEL%`
<div align="left">

```kts
player.configure {
  model = "%NO_MODEL%"
}
```

</div>
:::
