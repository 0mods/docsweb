::::info Сохранение
```kts
<players>.saveInventory()
```

Сохранить вы можете только 1 раз. После просто сохранение будет перезаписываться!
:::

:::info Загрузка
```kts
<players>.loadInventory()
```
:::

:::info Очистка :D
```kts
<players>.clearInventory()
```
:::
