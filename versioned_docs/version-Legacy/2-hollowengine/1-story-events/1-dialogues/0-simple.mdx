## Сказать

<br />

### НИП

:::info Сказать от лица НИПа абсолютно любой текст
```kts
<npcID> say { "<text>" }
```
---

```kts title="Пример"
npc say { "Привет" }
```
<Image path="doc/dialogue/npc_say.png" />

:::

<br />

### Игрок

:::info Сказать от лица игрока абсолютно любой текст
```kts
<player> say { "<text>" }
```
---

```kts
val me by server.players.filter { is.name.string == "_BENDY659_" }
```
```kts title="Пример"
ma say { "Привет" }
```
<Image path="doc/dialogue/player_say.png" />

:::
:::info Отправить личное сообщение игроку
```kts
<player> send { "<text?" }
```
---

```kts
val me by server.players.filter { is.name.string == "_BENDY659_" }
```
```kts title="Пример"
ma send { "Личное сообщение" }
```
<Image path="doc/dialogue/player_send.png" />

:::
