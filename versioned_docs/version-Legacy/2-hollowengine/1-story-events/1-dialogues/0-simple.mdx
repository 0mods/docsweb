## Сказать

<br />

### НИП

:::info Сказать от лица НИПа абсолютно любой текст
```kts
<npcID> say { "<text>" }
```
---
```kts title="Пример"
npc say { "Привет" }
```

![npc-say](@site/static/img/doc/dialogue/npc_say.png)

:::

<br />

### Игрок

:::info Сказать от лица игрока абсолютно любой текст
```kts
<player> say { "<text>" }
```
---
```kts title="Пример"
me say { "Здрасте" }
```

![player-say](@site/static/img/doc/dialogue/player_say.png)

:::
:::info Отправить личное сообщение игроку
```kts
<player> send { "<text?" }
```
---
```kts title="Пример"
me send { "Личное сообщение" }
```

![player-send](@site/static/img/doc/dialogue/player_send.png)

:::
