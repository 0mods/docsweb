# AnimationType

</div>

```kts
enum class AnimationType {
  IDLE,          // Стоять
  IDLE_SNEAKED,  // Стоять крадучись
  WALK,          // Идти
  WALK_SNEAKED,  // Идти крадучись
  RUN,           // Бежать
  SWIM,          // Плавать
  FLY,           // Летать
  FALL,          // Падать
  SIT,           // Сидеть
  SLEEP,         // Спать
  SWING,         // Удар рукой
  HURN,          // Урон (получен)
  DEATH          // Умер
}
```
