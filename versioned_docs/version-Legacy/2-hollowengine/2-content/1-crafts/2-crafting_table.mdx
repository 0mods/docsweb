## Форменный рецепт
:::info Форменный рецепт
<div align="left">

```kts title="Шаблон"
CraftingTable.shaped(
  // Предмет, получаемый при крафте
  item(
    itemID: String, // Id предмета
    count: Int, // Количество получаемого предмета
    nbtTags: String, // ТNBT-теги предмета
  )
) {
  // Тетка крафта 3x3
  grid(
    "[][][]",
    "[][][]",
    "[][][]"
    /*
     * За место [] нужно вставить:
     * - Почти любой символ
     * - Пробел, для пустой ячейки
    */
  )
  where {
    // Укажите символ в крафте и чему он равен
    '<key>' - item(itemId: String)
    ... // Можно несколько
  }
}
```
---
```kts title="Пример"
CraftingTable.shaped(item("name_tag", 4)) {
	grid(
		" S",
		"LI"
	)
	where {
		'S' - item("string")
		'L' - item("leather")
		'I' - item("ink_sac")
	}
}
```
