## Имеющиеся команды

::::info Команды

:::warning Все команды начинаются с
`/hollowengine`
:::

| Команда         | Аргументы                           | Действие                                                                        |
|-----------------|-------------------------------------|---------------------------------------------------------------------------------|
| `pos`           | -                                   | Получает точку позиции, куда вы в этот момент смотрели                          |
| `hand`          | -                                   | Получает предмет из вашей **главной руки** и выводит всю информацию о нём в чат |
| `model`         | путь: ResourceLocation              | Выводит информацию о модели, а именно: список анимаций и текстур                |
| `active-events` | -                                   | Выводит в чат список активных на данный момент скриптов                         |
| `start-script`  | цель: Игрок, путь: ResourceLocation | Запускает файл скрипта по указанного пути                                       |
| `stop-script`   | цель: Игрок, путь: ResourceLocation | Останавливает скрипт по указанному пути                                         |
| `compile-all`   | -                                   | Компилирует все скрипты из папки `scripts` для готовой работы                   |

::::
