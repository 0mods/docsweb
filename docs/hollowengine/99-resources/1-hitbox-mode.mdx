# HitboxMode

</div>

```kts
enum class HitboxMode {
  BLOCKING, // Блокирующий. Невозможно сдвинуть с места
  PULLING,  // Толкающий. Классический режим, когда сущность можно перемещать толкая
  EMPTY,    // Пустой. Отсутствие колизии
}
```
