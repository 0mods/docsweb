# `/HollowEngine команды`

Прописав в чате символ для команд `/` вы могли заметить совершенно новую команду

![cheater](@site/static/img/doc/titles/cheater.png "Читерюга")

---

## Имеющиеся команды

::::info Команды

:::warning Все команды начинаются с
`/hollowengine`
:::

| Команда         | Аргументы                           | Описание                                                                                                                   |
|-----------------|-------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| `pos`           | -                                   | Получает точку позиции, куда вы в этот момент смотрели                                                                     |
| `hand`          | -                                   | Получает предмет из вашей **главной руки** и выводит всю информацию о нём в чат                                            |
| `model`         | путь: ResourceLocation              | Указав путь к модели в формате `ResourceLocation` - выводит информацию о модели, а именно: список анимаций и слоёв текстур |
| `active-events` | -                                   | Выводит в чат список активных на данный момент скриптов                                                                    |
| `start-script`  | цель: Игрок, путь: ResourceLocation | Запускает файл скрипта из пути указанному в формате `ResourceLocation`                                                     |
| `stop-script`   | цель: Игрок, путь: ResourceLocation | Останавливает скрипт по указанному пути в формате `ResourceLocation`                                                       |
| `compile-all`   | -                                   | Компилирует все скрипты из папки `scripts` для готовой работы                                                              |

::::
