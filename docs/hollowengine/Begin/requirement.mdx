# Зависимости мода

:::caution Для работы мода `Hollow Engine` требуются такие моды, как:
```mermaid
graph TD;
  HE[HollowEngine]; HC[HollowCore]; KFF[KotlinForForge]; KS[KotlinScript];
  FTBT[FTB Team]; FTBL[FTB Library]; Arc[Architectury];
  
  HE-->|Требует|KS; HE-->|Требует|KFF; HE-->|Требует|HC; HE-->|По выбору|FTBT;
  KS-->|Требует|HC;
  FTBT-->|Требует|FTBL; FTBL-->|ЬТребует|Arc;
```
ℹ️ FTBTeams - не Требуемая библиотека! Устанавливайте, если хотите писать сюжет ℹ️

⚠️ Нужно устанавливать классическую версию HollowCore, без пометки(т.е. не **no-compiler**, **lite**, **original**) ⚠️
:::
