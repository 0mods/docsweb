# Commands in Hollow Engine

:::info Currently, the following commands exist:
- `/hollowengine pos` - Copies the position of the block you are looking at.
- `/hollowengine hand` - Copies the `itemID` of the item you are holding (also copies its NBT tags).
- `/hollowengine start-script <who> <path/to/file>` - Launches a script. Replace `who` with the intended user and `path/to/file` with the path to the script itself (must end with `.se.kts`).
- `/hollowengine model <modelFileName>` - Displays the animations and textures contained in the model.
- `/hollowengine active-event` - Activates a specific event.

:::
