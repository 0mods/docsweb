# Create a new item

:::caution Reminder
> Scripts in this category `Script - MOD` are written in `.mod.kts`!
:::

---

:::note Creating a New Item and Setting Its Attributes
```kts
import net.minecraft.item.Item
import net.minecraft.item.Food
import net.minecraftforge.registries.DeferredRegister
import net.minecraftforge.registries.ForgeRegistries
import ru.hollowhorizon.hollowengine.common.tabs.HOLLOWENGINE_TAB
// All the above are necessary imports.

val ITEMS = DeferredRegister.create(ForgeRegistries.ITEMS, "hollowengine") // You can add the item to any mod, for example, "hollowengine"

ITEMS.register("new_item") { // Creating a new item
    Item(
        Item.Properties() // Optional settings
        .food( // The item can be eaten. Now it's meat
            Food.Builder()
                .meat() // MEAT!
                .nutrition(6) // Nutritional value
                .saturationMod(0.7f) // Saturation percentage from nutrition
                .alwaysEat() // Can be eaten with full hunger bar
                .fast() // Can be eaten quickly
                .build() // Complete the food setup
        )
        .stacksTo(16) // Number of items in a stack
        .durability(10) // Item durability
        .tab(HOLLOWENGINE_TAB) // Add the item to the mod's tab
        .fireResistant() // Will not burn in lava
        .setNoRepair() // Cannot be repaired
    )
}

ITEMS.register(MOD_BUS) // Register our item
```
> Designations:

> `modID` - The mod's ID. In this case, I recommend using an existing modID as it is unknown how it will work with a new modID.
> `new_itemID` - The new item's ID.
> `ModID_TAB` - The creative tab of the mod. If you don't know the names of the tabs, I recommend using HOLLOWENGINE_TAB.

:::
