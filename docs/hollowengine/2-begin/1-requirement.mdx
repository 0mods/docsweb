:::caution 🔷 Для работы мода `Hollow Engine` требуются такие моды, как 🔷:
```mermaid
graph TD;
  HE[HollowEngine]; HC[HollowCore]; KFF[KotlinForForge]; KS[KotlinScript];
  FTBT[FTB Teams]; FTBL[FTB Library]; Arc[Architectury];
  
  HE-->|необходимо|KS;
  HE-->|необходимо|KFF;
  HE-->|необходимо|HC;
  HE-->|опционально|FTBT;

  KS-->|необходимо|HC;

  HC-->|необходимо|KFF;

  FTBT-->|необходимо|FTBL;
  FTBT-->|необходимо|Arc;
  FTBL-->|необходимо|Arc;
```
ℹ️ FTBTeams - опциональная библиотека! Устанавливайте, если хотите писать сюжет ℹ️

⚠️ Для написания скриптов нужно устанавливать полную версию KotlinScript (без пометкок вроде: **no-compiler**, **lite**, **original** и т.п.) ⚠️
:::
