# Диалоги

:::caution Перед началом
Если вы используете какого-то NPC, то диалог должен идти после того, как вы создали нужных NPC.
:::
## Начало диалога

:::info Чтобы начать диалог, нужно
```jsx
dialogue {
    // Здесь действия...
}
```
:::

### Действия

:::info Есть на данный момент эти действия
> - `<npcID> say "<text>"` - Слова от лица указанного NPC.
> - [Анимации NPC](../NPCs/actions-npc).
> - `team.send { "<text>" }` - Слова от лица главаря команды.
:::

---

## Выбор и последствия

:::info Выбор создаётся внутри `dialogue{...}`
```jsx
choice {
    "Ваш вариант 1" {
        // Последствия...
    }
    "Ваш вариант 2" {
        // Последствия...
    }
    // И так далее...
}
```
:::

---

## Сказать от кого-то

:::info Так же вы можете сказать от чьего-то лица

```jsx
    team.send {"<text>"} // Сказать от лица главоря команды
```
:::

:::info Или вы можете искать даже от не существующего персонажа

```jsx
    send { // Указывайте вручную: NAME - Имя, TEXT - текст, который нужно показать
	name = "<NAME>"
	text = "<TEXT>"
    }
```
:::

Пример скрипта:
```jsx
dialogue{
    npc say{"Привет пупсик!"}
    
    choice {
        "Привет пупсик" {
            npc.play{
                animation = "dance"
                layerMode = LayerMode.OVERWRITE
                playType = PlayType.LOOPED
            }
            npc say{"Ей, ДЕНС"}
            
            team.sendAsPlayer{"ДААА!!!"}
        }
        
        "Фу, уже неактуал" {
            npc say{"Фу, новомодный"}
            wait{5}
            
            npc say{"Уйди от меня"}
        }
    }
}
```
