# Асинхронные действия

:::note Действия в несколько потоков
> Появляется возможность выполнять несколько действий в 1 момент.
> Можно ставить друг за другом.
```jsx
// Данные действия будут работать в другом потоке
async{
    ...
}

// Если вам нужно ещё несколько, по просто идите по проядку
async{
    <actions1>
}
async{
    <actions2>
}
... // И так далее
```
:::

---

Пример:
```jsx
async{
    npc moveToPos{pos(10, 54, 23)}
}
nok setLookingEntity npc
```
