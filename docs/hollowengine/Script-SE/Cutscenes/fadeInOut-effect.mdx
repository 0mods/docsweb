# Screen Darkening Effect

:::info ...
```kts
fadeIn { // Fade in
    text = "<text1>" // Big text
    subtitle = "<text2>" // Small text below the big one
    texture = "<path/to/texture>" // Darkening texture
    color = "<0xHEX>" // Texture color
    time = <time> // Transition time
}

// Here you can put various actions, for example - Spawn NPC...

fadeOut { // Fade out
    text = "<text1>" // Big text
    subtitle = "<text2>" // Small text below the big one
    texture = "<path/to/texture>" // Darkening texture
    color = "<0xHEX>" // Texture color
    time = <time> // Transition time
}
```
> Designations:
> - `time` is specified in:
>  - `<num>` - Time in ticks.
>  - `<num>.sec` - Time in seconds.
>  - `<num>.min`, - Time in minutes.
>  - `<num>.hour`, - Time in hours.
>  - You can also combine them, using `+`, for example: `1.hour + 15.min + 30.sec + 45`.
> - `0xHEX` - Simple color in HEX encoding. You need to write 0x at the beginning and then only 5 digits.
:::
