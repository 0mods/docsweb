# Группы и групповые действия

## Создать группу

:::note Чтобы добавить НИПов (2 и более) в 1 группу
```jsx
val <groupID> = arrayOf(<npcID1, npcID2, npcID3, ...>)
```
:::

## Действия для группы

:::note Чтобы дать действие для группы
```jsx
<groupID.forEach{it <action>}>
// action - это те же действия что и для одного НИПа. Но т.к. это группа, мы даёт это действие каждому НИПу, кто находится в этой группе
```
> `action` - Те же действия, что и [для НИПов](actions-npc.mdx), по без `<npcID> ` в начале.

![Группа из НИПов идёт](../../.resourses/group-npcs-moveToPos.gif)

:::

---

Весь используемый скрипт:
```jsx
val kate by NPCEntity.creating{
  name = "Катя"
  model = "models:template/classic.gltf"
  textures["gltf_texture_template.classic.gltf_0"] = "textures:skins/black5.png"
  showName = false
  world = "overworld"
  pos = pos(0.5, -58.0, 0.5)
}
val ula by NPCEntity.creating{
  name = "Юля"
  model = "models:template/classic.gltf"
  textures["gltf_texture_template.classic.gltf_0"] = "textures:human2.png"
  showName = false
  world = "minecraft:overworld"
  pos = pos(3.5, -58.0, 8.5)
}
keybind{Keybind.H}

val girls = arrayOf(kate, ula)

girls.forEach{
  it moveToPos{pos(-13.5, -58.0, 6.5)}
}
```
