# Конфиг НИПа

:::info Всё перечисленное здесь, пишется...
```jsx
<npcID>.configure {
  // т.е. здесь!
}
```
:::

## Как работает

:::note Идём в "Спавним НИПа"
> По факту все тут настройки, это те же настройки что и при создании НПС, только тут ты можешь менять их динамично.
:::

---

Примеры скрипта:
```jsx
npc.configure{
    textures["based_texture"] = "textures:template/update_texture.png"
    textures["basic_skin"] = skin("_BENDY659_")
    animations[AnimationType.IDLE] = "betterIdle"
    
    transform = Transform(
        tx = 1f, tY = 2f, tZ = 1f,
        rX = 10f, rY = 50f, rZ = -84f,
        sX = 10f, sY = 5f, sZ = 6f
    )
}
```
