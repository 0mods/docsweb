# [В процессе обновления] Туториал №2 | Путь для камеры

:::note Создадим путь для перемещения камеры. Длительность пути будет 10 сек и установим интерполяцию `SINE_IN_OUT`
```kts
camera{
    time = 10.sec
    path = "camera_path.nbt"
    interpolation = Interpolation.SINE_IN_OUT
}
```
> [Использование камеры](https://youtu.be/VMMliA0TG2M)
:::
