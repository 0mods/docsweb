## Телепорт

:::info Телепорт

<div style={{"display": "flex"}}>
<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-move_to-pos](@site/static/img/doc/actions/npc-tp.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

```kts
<npcID> tpTo {
  pos = pos(x, y, z)
  vec = vec(pitch, yaw)
  world = "<dimentionID>"
}
```

Телепортирует НИПа на указанные координаты и в указанный мир (опционально).

> - `pos(x, y, z)` - Координаты, куда нужно телепортировать.  
> - `vec(pitch, yaw)` - Куда будет смотреть НИП после телепорта.  
> - `dimentionID` - Мир, в который нужно телепортировать НИПа. Если оно не указано - берётся тот мир,
в котором находится НИП в данный момент.

</div>
</div>

## Взаимодействие с миром

### Сломать блок

:::info Сломать блок

<div style={{"display": "flex"}}>
<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-move_to-pos](@site/static/img/null.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

```kts
<npcID> destroyBlock { pos(x, y, z) }
```

Сломать блок на указанных координатах.

НИП подойдёт к заданным координатам как можно ближе и после сломает блок.

---

```kts title="Пример"
npc destroyBlock { pos(10, 35, 221) }
```

## Атаковать

:::info Атаковать цель

<div style={{"display": "flex"}}>
<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-move_to-pos](@site/static/img/null.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

```kts
<npcID> setTarget { <target> }
```

НИП будет атаковать цель до последнего, пока цель не умрёт.

Целью может быть:  
> - Игрок - Нужно за место `<target>` указать
> <Random prct={70} prctMax={100}>
> <High>
> [**Определённого игрока**](@site/static/img/page-null.gif "Не кликай!").
> </High>
> <Low>
> [**Определённого игрока**](@site/static/img/page-null.png "Не кликай.").
> </Low>
> </Random>
> - Другой НИП - Нужно за место `<target>` указать `npcID` другого НИПа.
> - Сущность - Нужно за место `<target>` указать `Entity`. (IDK)

---

```kts
<npcID> setTarget { null }

// или

<npcID> clearTarget()
```

Перестать атаковать цель

---

```kts title="Пример"
npc setTarget { player().first() }

npc clearTarget()
```
