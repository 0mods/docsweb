# Двигаться

<Tabs>
<TabItem value="toPosition" label="До позиции" attributes={{className: tab.def}}>
<div style={{"display": "flex"}}>

<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-move_to-pos](@site/static/img/doc/actions/npc-move_to-pos.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

```kts
<npcID> moveTo { pos(x, y, z) }
```

НИП будет идти до указанных координат в `pos(x, y, z)`.

---

```kts title="Пример"
npc moveTo { pos(13, 65, 235) }
```

</div>
</div>
</TabItem>
<TabItem value="toPlayer" label="До игрока" attributes={{className: tab.def}}>
<div style={{"display": "flex"}}>

<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-move_to-player](@site/static/img/doc/actions/npc-move_to-player.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

```kts
<npcID> moveTo { <player> }
```

НИП будет идти до [**определённого игрока**](../../../hiddens/placeholder.mdx).

---

```kts title="Пример"
npc moveTo { player().first() }
```
</div>
</div>
</TabItem>

<TabItem value="alwaysMoveTo" label="Бесконечно" attributes={{className: tab.def}}>
<div style={{"display": "flex"}}>
<div atyle={{"margin": "20px", "align-items": "flex-start"}}>

![npc-always-move_to](@site/static/img/doc/actions/npc-always-move_to.gif)

</div>
<div align="left" style={{"margin": "10px", "align-items": "flex-end"}}>

Есть за место `moveTo` написать `alwaysMoveTo`, то НИП будет двигаться до указанного - бесконечно.

Для того чтобы остановить выполнение, нужно прописать:  
```kts
<npdID> stopAlwaysMove()
```

---

```kts title="Пример"
npc alwaysMoveTo { player().first() }

npc stopAlwaysMove()
```
