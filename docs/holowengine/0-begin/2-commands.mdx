# /HollowEngine Команды

<div className="show" id="mask">

![title](@site/static/img/doc/titles/Commands0.webp)

---

<br />

| Команда                      | Аргумент           | Описание                                                              |
|------------------------------|--------------------|-----------------------------------------------------------------------|
| `/hollowengine pos`          | -                  | Копирует позицию 3D-курсора (т.е. координаты блока, куда вы смотрели) |
| `/hollowengine hand`         | -                  | Копирует данные предмета (`item_id`, `count` и `nbt-tags`)            |
| `/hollowengine model`        | `ResourceLocation` | Выводит в чат инофрмацию о модели (Список анимаций и текстур)         |
| `/hollowengine start-script` | `<path/to/script>` | Запускает определённый скрипт                                         |
