# ↳ ▶ Конфиг НИПа ⚙️

:::info Всё перечисленное здесь, пишется
```kts
<npcID>.configure {
  // т.е. здесь!
}
```
:::

## Как работает

:::note Идём в "Спавним НИПа"
>  Поддерживаются почти все настройки, которые используются при [Создании НИПа](./0-npc-create.mdx)
:::

## Дополнительные настройки

:::note Дополнительные настройки
>  Все что написано в этой части, не относится в данному разделу, но подходит по смыслу.  
>  Все написанное здесь, не пишется в `<npcID>.configure{...}`.
> ```kts
> // Изменить имя НИПу
> 
> // Колизия хитбокса
> <npcID>.hitboxMode = HitboxMode.<Type>
> /* 
>  * Есть:
>  * - BLOCKING | Вы не сможете сместить нпс (будет стоять как камень)
>  * - PULLING  | Классический режим, когда подойдя к сущности вы отталкиваетесь друг от друга
>  * - EMPTY    | Без колизии. Проходить сквозь сущности без преград
> */
> 
> <npcID>.invulnerable = <Boolean: false/true> // Игнорирование урона НИПа
>
> <npcID>.name = "<String: newName>" // Новое имя НИПа
> ```
:::
