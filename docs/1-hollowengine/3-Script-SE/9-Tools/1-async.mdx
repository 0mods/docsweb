# ↳ ▶ Асинхронные действия ♊

:::note Действия в несколько потоков
> Появляется возможность выполнять несколько действий в 1 момент.
> Можно ставить друг за другом.
```kts
// Данные действия будут работать в другом потоке
async{
    ...
}

// Если вам нужно ещё несколько, по просто идите по прорядку
async{
    <actions1>
}
async{
    <actions2>
}
... // И так далее

// Так жы вы можете сделать async как переменные
val <id> = async{
...
}

// А после уже используя эти переменные, вы можете:
<id>.stop() // Остановить асинхронный процесс
<id>.resume() // Продолжить синхронный процесс
```
:::

---

Пример:
```kts
async{
    npc moveToPos{pos(10, 54, 23)}
}
nok setLookingEntity npc

val kerish = async{
    kola setLookingTeam{team}
}
wait{3.sec}

kerish.stop()
```
