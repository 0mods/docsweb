# Задержки в скриптах

## По таймеру

:::info Ты можешь поставить ожидание после каждого действия, перед тем, как перейти к следующему.
```jsx
wait{ <time> }
```
> `time`:
> - `<num>` - в тиках.
> - `<num>.sec` - в секундах.
> - `<num>.min` - в минутах.
> - `<num>.hour` - в часах.
> ---
> Так же вы можете соединять их через `+`, пример: `10 + 20.sec + 3.min + 5.hour`
:::

---

## По нажатию кнопки

:::info Скрипт будет ждать, пока не будет нажата указанная кнопка
```jsx
keybind { Keybind.<key> }
```
> `key` - Кнопки на клавиатуре. [Узнать больше...](https://github.com/HollowHorizon/HollowEngine/blob/1.19.2/src/main/java/ru/hollowhorizon/hollowengine/common/util/Keybind.kt)
:::

---

## По интерактиву

:::info Скрипт будет ждать, пока ты не по взаимодействуешь с определённым NPC
```jsx
<npcID>.waitInteract()
```
:::

## По тексту

:::info Скрипт продолжится, когда в чате появится нужное слово
```jsx
var text by input()
// В input() вы можете указать параметры, а именно - какие сообщения будут приняты -> input("Да", "Нет")
// Оставте данный параметр пустым, чтобы принималось лобое сообщение
```
:::

## По радиусу точки

:::into Скрипт продолжится, когда игрок(-и) окажется(-утся) в радиусе от заданной точки
```jsx
team waitPos{
    vec = pos(x, y, z) // Позиция
    radius = <double> // Радиус
    inverse = <false/true> // Инвертировать
    // Инвертировать омжет понадобится, если наоборот нужно чтобы игрок(-и) вышле(-и) из заданно радиуса в точке
}
```
:::
