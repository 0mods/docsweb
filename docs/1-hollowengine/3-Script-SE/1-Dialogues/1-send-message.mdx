# Сообщение в чат

## Для команды

:::info сообщение для всех игроков из команды
```kts
team.send {"<text>"}
```
:::

---

## От лица игрока

:::info Сообщение, адресатом которого будет главаря команды
```kts
team.sendAsPlayer {"<text>"}
```
:::
