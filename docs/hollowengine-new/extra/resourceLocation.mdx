# ResourceLocation

> Слова автора мода "HollowHorizon":
```text
- Путь к файлу указывается в виде "ResourceLocation".
Он предстваляет собой путь к файлу из определённого пространства имён. У каждого мода есть своё пространство имён.
Например `<modID>:path/to/file.ext`, указывает на файл `file.ext` расположенный по пути `path/to` в пространстве имён `modID`.
Сами пространсва имён - это папки в ресурспаках или самих модах и именуются они соотвественно самим модам.
```

---

Проще:

- после пути `hollowengine/assets/hollowengine/` Ты можешь размещать свои ресурсы в любом количестве папок в папке

    - Пример: `./assets/hollowengine/memory/dataFound/function/benOS/vitalik.txt`
