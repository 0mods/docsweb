# Special Dimension

:::info A new dimension stored in `hollowengine/storyteller_world`.
> An independent dimension that you can access from any world.
> To enter this dimension, you need a **special item**. You can obtain it from the `Hollow Engine` mod tab in the creative menu.
> Simply right-clicking will transport you to a new, empty place. There's only one block below you...
> ---
> To exit, simply right-click with the item again, and you will return to where you were.

:::
