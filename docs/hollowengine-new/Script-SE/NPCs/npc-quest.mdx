# Quests

:::info If you need to give something to the NPC, so the story can continue
```jsx
<npcID>.requestItems {
    // Write here, what you need to give to the NPC
}
```
:::

## Give items

:::note Give a particular item
```jsx
    +item("<itemID>", <count>) // Give [itemID] in the quantity [count]
```
> Designations:
> - `itemID` - Just take the desired item in your hand and use the command '/hollowengine hand', it will be copied, and you can paste it wherever needed.
> - `count` - Quantity of items.
:::
