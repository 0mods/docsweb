# Spawn NPC

## Template

:::info This is a template

```jsx
val <npcID> by NPCEntity.creating{
    settings = NPCSettings("<displayName>", "hollowengine:<path/to/model>.gltf",   Attributes(<attributes>), Pair(<float1>, <float2>), <boolean>)
    location = SpawnLocation("<dimension>", pos(<x>, <y>, <z>), vec(<pitch>, <yaw>))
}
```

- Designations:
> - `npcID` - Your NPC's ID. Through it, you address a specific NPC.
> - `displayName` - A name, shown above NPC's head. Just like a player's nickname.
> - `path/to/model` - Insert the path to the model.
> - `attributes` - Attributes for the NPC, for example: Health, speed and so on.
> - `Pair(<float1>, <float2>)` - NPC's hitbox.
>   - `float1` - Width.
>   - `float2` - Height.
> - `<boolean>` - Show `displayName` above the NPC's head.
>   - `false` - don't show,
>   - `true` - show.
> - `dimension` - The dimension, in which the NPC will spawn. You need to start with `modID`, for example: `minecraft:overworld`.
> - `pos(x, y, z)` - NPC's spawn position.
> - `vec(pitch, yaw)` - NPC's spawn angle.

- Unnessecary variables:
> - `attributes`. 
> - `Pair(),` 
> - `showName`.
:::

## Basic NPC

:::info To spawn a basic NPC, leave the field `NPCSettings()` - empty.
Here is an example, how the "Basic NPC" spawn script should look like:
```jsx
val npc by NPCEntity.creating{
    settings()
    location = SpawnLocation("minecraft:overworld", pos(0, -58, 0))
}
```

:::

## Custom NPC

:::info It is essential, that you specify the path to the model.
```jsx
val npc by NPCEntity.creating{
    settings("Vitalik", "hollowengine:models/entity/temp2.gltf")
    location = SpawnLocation("minecraft:overworld", pos(0, -58, 0))
}
```
![Custom NPC's Spawn](../../.resourses/spawn-custom-npc.gif)
:::

### Attributes

:::info Every existing attributes, you can find using the command `/attribute`
In your case, you only need to specify where `attributes`, is, which is:
```jsx
Attributes("<attributeName>" to <float>)
```
> Designations:
> - `attributeName` - name of the attribute.
> - `float` - Number/Digit, where in the end there is `f`, example: `10f`
:::
