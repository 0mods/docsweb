# Действия для NPC

## Передвижение

### Positon

:::info NPC будет двигаться до этих координат | Шаблон
```jsx
<npcID> moveToPos { pos(x, y, z) }
```
:::

### Team

:::info NPC будет двигать до ближайшего игрока в команде | Шаблон
```jsx
<npcID> moveToTeam { team }
```
:::

### Any NPC

:::info NPC будет двигать до другого NPC | Шаблон
```jsx
<npcID> movdToEntity { <any_npcID> }
```
:::

![NPC движется до точки](../../.resourses/npc-move.gif)

## Прочие действия

## Выкинуть предмет(бросить)

:::info Укажи NPC, какой предмет нужно выкинуть(бросить)
```jsx
<npcID> dropItem { item("<itemID>") } // На данный момент работает криво, кидает только на Юг. Ждём патча.
```
Предмет можно указать лёгким способом. Просто возьмите в основную руку нужный предмет и после пропишите команду `/hollowengine hand`, у вас скопируется данные о предмете вместе с `item`. Вам прсото нужно будет вставить это и всё.
:::

### Взять предмет в руку

:::caution
Для отображения предмета в руках у NPC. нужно модель, поддерживающая [Отображение предметов!](../../Tools/ItemHandVisible)
:::

#### Способ №1



:::info
Просто выкинуть ему предмет. Он подберёт его и возьмёт в правую руку.
:::

![NPC взял предмет в руку-1](../../.resourses/npc-give-item1.gif)


#### Способ №2

:::info Выдать ему определённый предмет в определённую руку через скрипт
```jsx
<npcID> giveLeftHand { item("<itemID>") } // Выдать предмет в левую руку
// or
<npcID> giveRightHand { item("<itemID>") } // Выдать предмет в правую руку
```
Предмет можно указать лёгким способом. Просто возьмите в основную руку нужный предмет и после пропишите команду `/hollowengine hand`, у вас скопируется данные о предмете вместе с `item`. Вам прсото нужно будет вставить это и всё.
:::
