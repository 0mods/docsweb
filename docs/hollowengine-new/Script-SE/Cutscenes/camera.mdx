# Камера

## Создать камеру

:::info Добавить камеру
```jsx
createCameraPath{}
```
:::

## Настройка камеры

:::info у камеры есть такие параметры как
```jsx
createCameraPath{
    spline(<time>,
        vec(<pitch_start>, <yaw_start>), vec(<pitch_end>, <yaw_end>)
        pos(x, y, z), // double. Если посте этого есть ежё позиции, то в конце этой строчки ставьте ",".
        interpolation = <interpolations>
        // И так сколькт вам нужно точек поворота
    )
}
```
> `time` указывается в:
> - `<num>` - Время в тиках.
> - `<num>.sec` - Время в секундах.
> - `<num>.min`, - Время в минутах.
> - `<num>.hour`, - Время в часах.
>
> Так же вы можете совмещать их, используя `+`, пример: `1.hour + 15.min + 30.sec + 45`.

> `interpolations` - [Список интерполяций](../Tools/interpolations)

> Что за что отвечает:
> - `time` - Устанавливает общее время движение камеры.
> - `interpolation = <interpolations>` - Устанавливает интерполяцию движения. (Проще говоря, ускаряем или замедляет в определённые мометы в зависимости от выбранной интерполяции).
> - `vec(<pitch_start>, <yaw_start>), vec(<pitch_end>, <yaw_end>)` - Устанавает поворот камеры в начале (`_start`) и в конце (`_end`).
> - `pos(x, y, z)` - Устанавливает точку позиции. Работает только с нецелыми числами.
:::
