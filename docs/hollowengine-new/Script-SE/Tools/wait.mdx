# Таймер

:::info Ты можешь поставить ожидание после каждого действия, перед тем, как перейти к следующему.
```jsx
wait{ <time> }
```
> `time`:
> - `<num>` - в тиках.
> - `<num>.sec` - в секундах.
> - `<num>.min` - в минутах.
> - `<num>.hour` - в часах.
> ---
> Так же вы можете соединять их через `+`, пример: `10 + 20.sec + 3.min + 5.hour`
:::
