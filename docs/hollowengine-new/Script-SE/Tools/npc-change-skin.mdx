# Смена текстуры / скина у NPC

:::info Сметить текстуру / скин можно таким образом | Шаблон
```jsx
<npcID>.configure {
    ["<texture>"] = "<new texture>" // На данный момент работает криво. Изменяется текстура после перезаргузки ресурсов на [F3 + T]
}
```
> Обозначения:
> - `texture` - Текстура, которая на данный момент у NPC.
> - `new texture` - Путь к новой текстуре. `hollowengine:<path/to/texture>`.
:::
