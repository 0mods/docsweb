# Передвижение NPC

За чем стоять, если можно - двигаться:

```jsx
<npcID>.makeTask{
    moving.to(<x>, <y>, <z>).<taskMode>
    }
```

Информация:

- `<npcID>` - это имя NPC для кода. У каждого NPC должен быть уникальное имя. Нужен для спавна NPC,

- `<taskMode>` - Условия выполнения. Информация -> [Режимы задач](../extra/taskMode).

У меня вышло так:

```jsx
vit.makeTask{
    moving.to(8, -59, -5).await()
    }
```
