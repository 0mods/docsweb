# Новая папка

При первом запуске мода, в директории появиться новая папка с именем `hollowengine`.

В ней будут 2 новые папки:
> - `scripts` - В ней вы будете хранить все свои обычные, и скомпилированные скрипты.
> - `storyteller-world` - В этой папке находиться новый мир.
