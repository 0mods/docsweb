# Виды скриптов

> Существует на данный момент несколько типов:

## SE | `<name>.se.kts`
:::note Сюжетный скрипт
> Это базовый скрипт который позволяет работать вам с: Триггерами, Ивентами, NPC's, и т.д.
:::

***

## HSD | `<name>.hsd.kts`
:::note Диалоги с выбором
> Этот тип скрипта нужен для спец. диалоговых окон.
:::

***

## MOD | `<name>.mod.kts`
:::note Создаём свой мод
> Этот тип скрипта поможет вас создать свой полноценный мод для сюжета.
:::

***

## CONTENT | `<name>.content.kts`
:::note Изменяем крафты
> Этот тип скрипта поможет вам работать с крафтами предметов, а именно: добавлять, изменять, удалять крафты.
:::
