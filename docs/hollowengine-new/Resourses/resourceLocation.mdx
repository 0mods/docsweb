# Ресурсы

---

## Расположение ресурсов

:::caution Слова автора мода "HollowHorizon":
```text
- Путь к файлу указывается в виде "ResourceLocation".
Он представляет собой путь к файлу из определённого пространства имён. У каждого мода есть своё пространство имён.
Например `<modID>:path/to/file.ext`, указывает на файл `file.ext` расположенный по пути `path/to` в пространстве имён `modID`.
Сами пространства имён - это папки в ресурспаках или самих модах и именуются они соответственно самим модам.
```
:::

:::info Проще:

- Ты можешь располагать свои ресурсы в любом месте после `hollowengine/assets/hollowengine/`
:::

:::caution
- Если ты не сильно разбираешься в этом, то советую располагать ресурсы так:
  - `hollowengine/assets/hollowengine/models/<type>` - Здесь ты можешь располагать свои модели по категориям:
    > `item`, `block`, `entity` (скорее всего ЭТИ категории тебе пригодятся).

  - `hollowengine/assets/hollowengine/texturess/<category>` - Здесь ты можешь хранить свои текстуры по категориям:
    > `item`, `block`, `entity`, ... (скорее всего ЭТИ категории тебе пригодяться).
:::
