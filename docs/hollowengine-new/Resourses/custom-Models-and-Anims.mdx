# Кастом модели и анимации

> Ты можешь так же загружать и использовать Свою модель с анимациями.

:::info Тебе всего лишь надо:

- Создать в `/hollowengine/*` создать новые разделы `/assets/hollowengine`

    - Для тех кто не понял: В папке `hollowengine` создаёте новую папку с названием `assets`, после уже в ней создаёте ещё одну папку `hollowengine`.
:::

:::tip
```txt
    hollowengine
     |- assets
     |   \- hollowengine
     |- scripts
     \- storyteller_world
```
:::

Но если для вас это трудно то рекомендую:

> Хранить модель в пути -> `hollowengine/assets/hollowengine/models/entity/*`,

> Текстуру к модели в пути -> `hollowengine/assets/hollowengine/textures/entity/*`.
