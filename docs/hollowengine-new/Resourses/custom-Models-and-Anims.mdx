# Внешний вид NPC

## Модель

:::tip тип модели
- На данный момент поддерживаются модели `GlTF`.
:::

:::info где можно сделать?
Такую модель вы можете создать к примеру в **Blockbench** из простых программ.
:::

:::caution Информирую
- Модель и текстуры ты можешь хранить где угодно, но после `hollowengine/assets/hollowengine/`.

> Но если ты не разбираешься, то рекомендую хранить в этих местах:
>  - `hollowengine/assets/hollowengine/models/entity/<modelName>.gltf` - Здесь ты можешь хранить модель для NPC,
>  - `hollowengine/assets/hollowengine/textures/entity/<textureName>.gltf` - Здесь ты можешь хранить текстуру для моделей NPC.
:::

## Анимации для NPC
:::info Где делать анимации
- Анимации делаются там же где и модель, но слегка в другом месте.
:::

:::tip Название у анимаций
- Название может быть каким угодно.
:::
