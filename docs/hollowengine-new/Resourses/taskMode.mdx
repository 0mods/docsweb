# Режимы задач

> Это условие которое говорит движку что делать, пока идёт задача для NPC.

:::tip Виды:

- `.await()` - Скрипт дальше не пойдёт, пока NPC не завершит текущую задачу.

- `.async()` - Скрипт пойдёт дальше вместе с задачей. Игнорируется завершение NPC задачи.
:::
