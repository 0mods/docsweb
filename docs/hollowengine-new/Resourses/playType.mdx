# Типы воспроизведения анимаций

Все типы анимации пишутся в начале так:

```jsx
playType = *
```

"*" - не надо ставить.

:::info После выбираете какой тип вам нужен:

- `PlayType.ONCE` - Проигрывает анимацию 1 раз.

- `PlayType.LOOPED` - Проигрывает анимацию бесконечно раз. | Для прекращения требует `<npcID>.stop("<animName>")`

- `PlayType.LAST_FRATE` - Проигрывает анимацию 1 раз, после замораживается на последне кадре анимаиции. | Для прекращения требует `<npcID>.stop("<animName>")`

- `PlayType.REVERSED` - Проигрывает анимацию сначала до конца, после в обратную стону.
:::
