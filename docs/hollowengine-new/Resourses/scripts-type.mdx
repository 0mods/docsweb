# Типы скриптов

---

## SE | `<name>.se.kts`

> Является базовым и универсальным.

:::tip С его помощью вы можете

- Создать NPC,

- Взаимодействовать с NPC,

- Создать ивент в сюжете,

- Создать триггеры.
:::

---

## MOD | `<name>.mod.kts`

> Что-то типо Minecraft Mod Maker.

:::tip С его помощью вы можете
- Создать полноценный мод.
:::

---

## CONTENT | `<name>.content.kts`

> Можно работать с рецептами крафтов. Работает на основе Datapacks(Датапаки) и по идее должен работать со всеми модами.

:::tip С его помощью вы можете

-  Добавить / Изменить / Удалять рецепты крафтов.
:::

---

## HSD | `<name>.hsd.kts`

> Нужен для диалогов с NPC в спец. окне.

:::tip С его помощью вы можете:

- Создавать диалог

  - Делать выбор от которого будет зависить, как пойдёт ваш сюжет дальше.

  - `<invalid.reference>`
:::
