# Таймер

---
## Устанавливаем таймер

:::note Это таймер
Благодаря этому, ты устанавливаешь задержку перед выполнением следующего действия | Макет скрипта
```
wait(<count>f)
```
Обозначения:
> - `<count>f` - Время указывается в `сек`. Всегда пишется вместе с `f` в конце.

:::

***

Пример использования | Можете скопировать для проверки
```jsx
vitalik.makeTask{
    movement.go(14, -55, 1).await()
    )
wait(3f)

vitalik.play("addAnim.eyeLooking", priority=5f, playType=PlayType.ONCE)
```

![Виталик ждёт действия](../../../static/img/he-guide/NPC-Wait1.gif)

`В демонстрации используется Своя модель. Вы не сможете её никак использовать т.к. в файлах нет данной модели.`

---

## по ПКМ

:::info Если...
значение `wait(<значение>f)` оставить пустым, т.е. `wait()`, то скрипт продолжиться только после нажатия ПКМ.

Пример скрипта | Можешь скопировать для проверки
```jsx
vitalik.makeTask{
    look.at(team.getPlayer("<me>>").mcPlayer!!).await()
}
wait()

vitalik.play("addAnim.IDK", priority=5f, playType=PlayType.ONCE)
wait()
```

![Ожидаем ПКМ](../../../static/img/he-guide/NPC-Wait2.gif)

`В демонстрации используется Своя модель. Вы не сможете её никак использовать т.к. в файлах нет данной модели.`
