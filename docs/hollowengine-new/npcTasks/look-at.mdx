# Взгляд NPC

```jsx
<npcID>.makeTask{
    look.at(<type>).<taskMode>
```

> Инфо:

- `<npcID>` - это имя NPC для кода. У каждого NPC должен быть уникальное имя. через него ты будешь с ним взаимодействовать,

- `<taskMode>` - Условия выполнения. Информация -> [Режимы задач](../extra/taskMode),

- `<type>`:
  - `<x>, <y>, <z>` - При указании координат, взгляд NPC будет направлен в эту точку. Цифра / Число должно быть - Целое,

  - `<npcID>` - При указании другого NPC, взгляд NPC буде тнаправлен на него,

  - `<player>` - | `team.getPlayer("<playerNick>").mcPlayer!!` | <- Это тебе надо вписать. При указании Игрока, взгляд NPC будет направлен на него.
