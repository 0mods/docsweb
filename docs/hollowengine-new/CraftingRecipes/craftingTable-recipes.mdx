# Crafting Table recipes | Рецепты для Верстака (Стандартный 2х2 и 3х3)

:::tip Важно
Для работы потребуется другой тип скрипта: `<name>.content.kts`
:::

---

:::info Как узнать всё о предмете
В игре, берёте в руку нужный вам предмет и прописываете команду `/hollowengine hand`.
Вы узнаете всю информацию о предмете.
:::

---

```jsx
CraftingTable.removeByOutput(item("<item>")
```
- С помощью этого скрипта, ты сможешь Удалить рецепт.

:::info Инфо
- `<item>` - Предмет в игре, а именно его ID, пример: `gold_carrot`.  Начинается с `<modID>:*`.
:::

Вот пример как должно выглядеть:
```jsx
CraftingTable.removeByOutput(item("minecraft:crafting_table")
```

---

```jsx
CraftingTable.replaceShaped(item("<item_Out>", <count>, "<NBT_Tag>")) {
    grid(
        "[][][]",
        "[][][]",
        "[][][]"
    )

    where {
        `<simbol>` - item("<item_In>")
    }
}
```
- С помощью этого скрипта, ты можешь Добавить рецепт предмета.

:::info Инфо
- `<item>` - Предмет в игре, а именно его ID, пример: `gold_carrot`.  Начинается с `<modID>:*`,

- `<item_Out>` - Предмет получаемый при крафте / Предмет выхода,

- `<count>` - Кол-во выходного предмета,

- `<NBT_Tag>` - Для тегов у предмета если вам это нужно, пример "{Damage:15}",

- `[]` - Вместо этого поставьте Любой символ, к пример: @,

- `<simvol>` - Любой символ, который используется в схеме крафта, пример: @,

- `<item_In>` - Предмета, который будет использоваться к крафте.
:::

Вот пример как это должно выглядеть:
```jsx

CraftingTable.replaceShaped(item("minecraft:chest")) {
    grid(
        "@@@",
        "@!@",
        "@@@"
    )

    where {
        '@' - item(tag("minecraft:planks"))
        '!' - item("minecraft:iron_nuggets")
    }
}
```

Да, рецепты так же работают с тегами. Достаточно просто в `item()` вписать `tag(<item>)`.


---

```jsx
CraftingTable.shapeless(item("<item_Out>", <count>, <NBT_Tag>)) {
    items(
        item("minecraft:<item_In>")
    )
}
```
- С помощью этого скрипта ты сможешь создать бесформенный рецепт предмета.

:::info Инфо
- `<item_Out>` - Предмет получаемый при крафте / Предмет выхода,

- `<count>` - Кол-во выходного предмета,

- `<NBT_Tag>` - Для тегов у предмета если вам это нужно, пример "{Damage:15}",

- `<item_In>` - Предмета, который будет использоваться к крафте.
:::

Вот пример как должен выглядеть:
```jsx
CraftingTable.shapeless(item("minecraft:diamond_block")) {
    items(
        item("minecraft:diamond"),
        item("minecraft:dirt")
    )
```
