# Furnace recipes | Рецепты для Печки

:::tip Важно
Для работы потребуется другой тип скрипта: `<name.content.kts>`
:::

---

:::info Как получить информацию о требуемом предмете
В игре, берёте в руку нужный вам предмет и прописываете команду `/hollowengine hand`.
Вы узнаете всю информацию о предмете.
:::

---

```jsx
Furnace.removeRecipt("<item>")
```
- С помощью этого скрипта, ты сможешь Удалить рецепт.

:::info Инфо:
 - `<item>` - Предмет игре, а именно его ID, пример: `gold_carrot`.
:::

Вот пример как должно выглядеть:
```jsx
Furnace.removeRecipt("minecraft:iron_ingot_from_smelting_deepslate_iron_ore")
```

---

```jsx
Furnace.addRecipe("<item_Out>", "<item_In>", exp:<count>, time:<count>)
```
- С помощью этого скрипта, ты сможешь Добавить рецепт.

:::info Инфо 
- `<item_Out>` - Предмет в игре, а именно его ID, пример: `gold_carrot`.  Начинается с `<modID>:*`,


- `<item_In>` - Предмет в игре, а именно его ID, пример: `gold_carrot`.  Начинается с `<modID>:*`,

- `exp:<count>` - Опыт, получаемы после переплавки. Если отсутствует - Опыт не будет получен,

- `time:<count>` - Время переплавки предмета.
:::

Вот пример как должен выглядеть:
```jsx
Furnace.addRecipe("minecraft:diamond_block", "minecraft:diamond", exp:10, time:10)
```
