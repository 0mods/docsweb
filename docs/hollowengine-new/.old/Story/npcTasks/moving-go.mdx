# Движение NPC

Благодаря этой задаче ты можешь поручить NPC дойти до нужной точки. Но он не встанет ровно по центру блока.

```jsx
<npcID>.makeTask {
    movement.to(<x>, <y>, <z>).<taskMode>
}
```
:::info Инфо

- `<npcID>` - это имя NPC для кода. У каждого NPC должен быть уникальное имя. через него ты будешь с ним взаимодействовать,

- `<taskMode>` - Условия выполнения. Информация -> [Режимы задач](../../taskMode),

- `<x>, <y>, <z>` - Координаты. Цифра / Число должно быть - Целое.
:::

Пример:
```jsx
npc.makeTask{
    momevent.go(8, -59, -5).await()
    }
```
