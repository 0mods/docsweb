# [ Not Working ] Чёрный экран

С помощью этого скрипта, ты сделаешь чёрный экран(а синий лучше):

```jsx
makeTransition("displayText", <number>f) {
    <addScripts>
    }
```

:::info Инфо:
- `<displayText>` - Текст, который будет виден на чёрном экране.

- `<number>` - любая(-ое) цифра / число.

- `f` - float. обязательно должно быть.
:::
