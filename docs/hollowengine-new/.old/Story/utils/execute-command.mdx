# Execute | Использование игровых команд через `/`

---
:::caution Импортируй библиотеку
```jsx
import  ru.hollowhorizon.hollowengine.common.scripting.story.extensions.*
```
Без неё у тебя будет ошибка.
:::

---

Есть вещи которые надо делать через обычные команды, но не удобно каждый раз вводить их вручную, особенно если это одна и та же команда.
```jsx
execute("/<command>")
```

:::info Инфо
- `/<command>` - Обычная команда, пример: /setblock.
:::
