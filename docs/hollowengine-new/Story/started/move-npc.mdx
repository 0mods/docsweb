# Передвижение NPC

За чем стоять, если можно - двигаться:

```jsx
<npcID>.makeTask{
    moving.to(<x>, <y>, <z>).<taskMode>
    }
```
:::info Инфо

- `<npcID>` - это имя NPC для кода. У каждого NPC должен быть уникальное имя. через него ты будешь с ним взаимодействовать,

- `<taskMode>` - Условия выполнения. Информация -> [Режимы задач](../../Resourses/taskMode).
:::

У меня вышло так:

```jsx
vit.makeTask{
    moving.to(8, -59, -5).await()
    }
```
