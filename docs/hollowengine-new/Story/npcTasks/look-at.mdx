# Взгляд NPC

```jsx
<npcID>.makeTask{
    look.at(<type>).<taskMode>
```

:::info Инфо
- `<npcID>` - это имя NPC для кода. У каждого NPC должен быть уникальное имя. через него ты будешь с ним взаимодействовать,

- `<taskMode>` - Условия выполнения. Информация -> [Режимы задач](../../Resourses/taskMode),

- `<type>` - Параметр задачи:

  - Если тебе надо, чтобы NPC посмотрел на определённый блок, достаточно в параметре указать координаты блока: `<x>, <y>, <z>`,

  - Чтобы NPC посмотрел на другого NPC, надо в параметре указать ID этого NPC:`<npcID>`,

  - Если надо, чтобы NPC посмотрел на главного в команде, надо указать это: `team`,

  - Если тебе надо, чтобы NPC посмотрел на определённого игрока, надо ввести в параметр это:`team.getPlayer("<playerNick>").mcPlayer!!`, где в `<playerNick>` надо указать имя этого игрока.
:::
