:::note actions
> We spawn a Basic NPC, wait for 2 seconds, tell him to go to a certain point (position), then looks towards the player, after 3 seconds it dissapears.

```jsx
val npc NPCEntity.creating{
    settings = NPCSettings()
    location = SpawnLocation("minecraft:overworld", pos(0, -58, 0))
}
wait { 2.sec }

npc moveToPos { pos(-10, -58, 6) }
npc lookAtTeam { team }
wait { 3.sec }

npc.despawn()
```
