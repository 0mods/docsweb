## Установке текстуры / скина

:::note Установить новую текстуру
```jsx
  textures["<texture>"] = "<path/to/new_texture>" // Заменяем текущую текстуру на навую
```
> Обозначения:
> - `texture` - Имя текстуры, которая сейчас на модели.
> - `<path/to/new_texture>` - Путь к новой текстуре. Начинаться должен с `<modID>:`.
>  - `modID` - `ID` мода.
:::

---

:::note Установить новый скин
```jsx
  textures["<texture>"] - skin("<nickname>") // Работает только с ОФИЦИАЛЬНЫМИ скинами Mojang. Вводите корректно имя, иначе -> краш
```
> Обозначения:
> - `texture` - Имя текстуры, которая сейчас на модели.
> - `nickname` - Ник игрока, у которого есть лицензия.
:::

## Установка стандартных анимаций

:::note Замени стандартную анимацию
```jsx
  // Замена стандартной анимации на новую
  animation[AnimationType.<type>] = "<new_animationName>"
```
> Обозначения:
> - `type` - [Тип анимации](../../Tools/standartAnimationType)
> - `new_animationName` - Название анимации
:::

---

## Установить трансформацию модели

> Изменить положение, поворот, размер

:::note Все изменения происходят:
```jsx
  transform = Transform(
      // Тут!
  )
```
:::

:::note Изменить положение
```jsx
      tX = <float>, // Изменить положение по X
      tY = <float>, // Изменить положение по Y
      tZ = <float>  // Изменить положение по Z
```
:::

:::note Изменить поворот
```jsx
      rX = <float>, // Изменить поворот по X
      rY = <float>, // Изменить поворот по Y
      rZ = <float>  // Изменить поворот по Z
```
:::

:::note Изменить размер
```jsx
      sX = <float>, // Изменить размер по X
      sY = <float>, // Изменить размер по Y
      sZ = <float>  // Изменить размер по Z
```
:::

---

## Изменить ось поворота головы

:::note Если голова поворачивается не в ту сторону, куда направлен взгляд
```jsx
  // Если голова поворачивается не правильно
  switchHeadRot = <false/true>
```

---

Примеры скрипта:
```jsx
npc.configure{
    textures["based_texture"] = "textures:template/update_texture.png"
    textures["basic_skin"] = skin("_BENDY659_")
    animation[AnimationType.IDLE] = "betterIdle"
    
    transform = Transform(
        tx = 1f, tY = 2f, tZ = 1f,
        rX = 10f, rY = 50f, rZ = -84f,
        sX = 10f, sY = 5f, sZ = 6f
    )
}
```
