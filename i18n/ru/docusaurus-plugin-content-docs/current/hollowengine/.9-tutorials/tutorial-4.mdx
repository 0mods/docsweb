# Туториал №4 | Создание простых функций

> Для того, что не писать огромное кол-во одинаковых раз строк, можно упростить себе задачу и запихнуть всё в 1 функцию, а после в скриптах вызывать её в нужный момент.

:::info Шаблон

```kts
fun <FucntionName>(<Parameters>) {
    <FunctionBlock> // То, где будут происходить все вычисления
}
```

:::

---

## Простая математика

:::note Предположим у нас есть такой скрипт

```kts
val npc by NPCEntity.creating{
    name = "Юля"
}

val num1 = input() // Принимает любое сообщение от игрока
val num2 = input() // Принимает любое сообщение от игрока

npc say{"$num1 + $num2 = ${<Функция, где на выходе будет ответ>}"}
```

---

> Мы можем под нашим скриптом создать функцию, к примеру
```kts
fun mathOnNPC(numberOne: String, numberTwo: String): Int {
    return numberOne.toInt() + numberTwo.toInt()
}

/*
 * Что же мы сделали?
 * > Создали функцию с названием mathOnNPC(), которая принимает параметры:
 *  - Параметр numberOne со значением String (текст, строка)
 *  - Параметр numberTwo со значением String (текст, строка)
 * - Данная функция позращает значение Int (целое число)
 *
 * > В блоке функции происходит:
 *  - Мы возращаем сумму numberOne И numberTwo
 *  - Т.к. это сложение, мы не можешь принимать буквы и возращать нам нужно число.
 *     Поэтому мы переводим String -> Int с помощью функции String.toInt()
 *     [ .toInt() ]
*/
```

---

> После же наш скрипт будет выглядеть как-то так
```kts
val npc by NPCEntity.creating{
  name = "Юля"
}

val num1 = input()
val mun2 = input()
val result = mathOnNPC(num1, num2)

npc say{"$num1 + $num2 = $result"}

fun mathOnNPC(numberOne: Int, numberTwo: Int): Int {
    return numberOne.toInt() + numberTwo.toInt()
}
```

---

> Итого, если мы отправим в чат сообщения, где будут цифры к примеру 15 и 45, то мы получим это:
```txt
[Юля] 14 + 45 = 60
```

:::
