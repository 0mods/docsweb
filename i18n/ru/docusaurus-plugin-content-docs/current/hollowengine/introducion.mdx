# Введение

HollowEngine - это многофункциональный движок, который позволяет создавать разнообразные сюжеты в игре. Но это не единственная его функция. Он предоставляет множество возможностей для моддинга, не ограничиваясь одним только созданием NPC.

На данный момент HollowEngine предоставляет простой и удобный способ создания скриптов на языке Kotlin. В будущем появятся и другие возможности, такие как визуальные редакторы и команды.

Если у вас есть какие-либо вопросы по моду, вы можете обратиться к нам на [Discord сервере](https://discord.gg/qKpPhkwGCY).

# С чего начать?

В данный момент вы можете начать создавать различные скрипты для игры. Для более подробной информации обратитесь к главе [Скрипты: Введение](.scripting/introducion)
