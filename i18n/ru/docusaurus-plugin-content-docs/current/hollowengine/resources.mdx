# Ресурсы HollowEngine

В ваших проектах вам зачастую понадобится добавлять свои ресурсы, будь то модели, текстуры, файлы перевода и т.п.
Поэтому в HollowEngine есть 2 специальные папки для ресурсов.

## Клиентские Ресурсы

Ресурсы хранящие всякие данные для клиента, можно сказать "Ресурс-паки". Здесь хранятся все модели, текстуры и прочие клиентские ресурсы.
Их нужно хранить по пути: `(папка с игрой)/hollowengine/assets/(название мода)/`.

Название мода - мод которому будут присовоены эти ресурсы, т.е. по сути вы можете заменять текстуры и модели других модов, так и добавлять свои собственные. Рекомендуем использовать название "hollowengine".

Данные ресурсы обязаны быть у каждого игрока сервера/сборки.

## Серверные Ресурсы

Ресурсы хранящие всякие данные для сервера, т.е. "дата-паки". Здесь хранятся теги предметов, рецепты, структуры и т.п.
Их нужно хранить по пути: `(папка с игрой)/hollowengine/data/(название мода)`.

Аналогично с клиентскими ресурсами, здесь можно заменить или добавить новые ресурсы к конкретному моду.

Данные ресурсы не обязательны для клиента, их можно не добавлять в клиентскую сборку (если вы разрабатываете сервер).

## Примечания

- При изменении клиентских ресурсов для того чтобы они обновились в игре вам необходимо нажать сочетание клавишь `F3 + T`, чтобы игра перезагрузила наборы ресурсов.
- При изменении серверных ресурсов необходимо прописать команду '/reload'.
- Вы также можете использовать другие моды-загрузчики ресурсов вроде Open Loader или KubeJS.
