## Запуск Сюжетного события
:::tip Немного о запуске
Для запуска сюжетного события введите `/hollowengine start-script`, после следующим аргументом передайте `игнорировать ли пройденные события` (true/false),
если событие уже пройдено, то оно не запустится без `true` аргумента и далее, следующим аргументом найдите в списке ваше событие.
Если после названия указать имя игрока или тег типа @a, то диалог запустится у конкретного игрока/группы.
:::
