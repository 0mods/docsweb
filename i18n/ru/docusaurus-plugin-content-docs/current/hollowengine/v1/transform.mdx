# 2.3 | Диалоги | Трансформация
Трансформация объектов в диалогах

# Application
Трансформацию можно применять на различные объекты, на данный момент это: Персонажи и Картинки.
Соответственно в примерах `object` может означать как Персонажа, так и картинку, и там, и там это будет работать.

# Размер
Иногда бывает, что какой-то персонаж в диалоге слишком большой или маленький, например если моб - Чешуйница или Эндер Дракон. :)
Для того, чтобы изменить это значение, напишите:

`object.scale = 0.5f` - где присваивается значение размера моба.

# Permutation
Если нужно переместить объект на некоторое расстояние, или сделать небольшую анимацию (подробнее распишу позже)
типа дрожания/ходьбы из стороны в сторону, то проще всего использовать:

`object.translate = Vector2f(x, y)` - не забудьте в начале файла добавить `import net.minecraft.util.math.vector.Vector2f`
иначе будет ошибка, x и y, это соответственно перемещение по x и по y, очевидно, не так ли?
