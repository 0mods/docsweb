# 1.2 | Скриптовой движок | Расширение файлов

|         Тип         | Расширение |                                         Описание                                         |   Комментарий   |
|:-------------------:|:----------:|:----------------------------------------------------------------------------------------:|:---------------:|
|         Мод         |  mod.kts   | Загружается при запуске игры. Позволяет сделать то же самое, что можно сделать и в моде. |   Реализовано   |
|  Сюжетное событие   |   se.kts   |   Загружается при заходе в мир. Позволяет создавать всякие триггеры, ивенты и другое.    |   Реализовано   |
|       Диалоги       |  hsd.kts   |                 Загружается при заходе в мир. Позволяет делать диалоги.                  |   Реализовано   |
|       Рецепты       | recipe.kts |                Позволяет создавать свои и изменять существующие рецепты.                 | Разрабатывается |
| Spongepowered Mixin | mixin.kts  |                        Позволяет менять код игры прямо в скрипте.                        | Разрабатывается |
