# 2.2 | Диалоги | Анимация

## Создание простых скриптовых анимаций
Принцип работы любой анимации - изменение состояния объекта (моба/картинки) с течением времени. Соответственно,
если вы например хотите сделать анимацию увеличения персонажа, то это делается так:
```kotlin
for(i in 0..100) {
    object.scale = 1.0f + i / 100f //add the current size to the original size, because if i=50 it will be: 1f + 0.5f, at i=75: 1f + 0.75f, etc.
    wait(0.1f) // wait 1/10 seconds
}
```

Это самый примитивный способ, что-либо анимировать, таким же образом можно и [перемещать](./transform) персонажей на экране.
Например, если перемещать каждую 10 долю секунды персонажа на случайное значение по x и по y,
то можно сделать анимацию тряски и тому подобное. Вполне вероятно в будущем будет более простой способ создания подобных анимаций,
с интерполяцией в пару строк :)
