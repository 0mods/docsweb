# Скрипты: Моды

## Описание

Данный вид скрипта предназначен для различных целей, которые необходимо выпустить до загрузки игры:
- добавить игровой контент: блоки, предметы, сущности, измерения и т.п;
- зарегистрировать определённые Forge или ModBus события;
- создать какие либо конфиги.

## Переменные

### События

`FORGE_BUS` - переменная содержащая обработчик событий Forge. Аналогична вызову `MinecraftForge.EVENT_BUS`.
`MOD_BUS` - переменная содержащая обработчик событий мода. Аналогична вызову `FMLJavaModLoadingContext.get().getModEventBus()`.

:::info Пример обработки Forge события
```jsx
import net.minecraftforge.client.event.ScreenEvent

//При открытии какого-либо интерфейса
fun onScreenOpen(event: ScreenEvent.Opening) {
	//Тут ваш код обработки события
}

//Передача вашего события в обработчик событий Forge
FORGE_BUS.addListener(this::onScreenOpen)

```
:::

### Прочее
`dist` - сторона на которой запущен код, физический клиент или сервер.

:::info Пример использования
```jsx
if(dist.isClient) {
	//Если скрипт запущен на клиенте
}

```
:::

## Примечания
- ModBus события привязываются к моду HollowEngine, так что учитывайте это при разработке.
- Скорее всего в недалёком будущем mod-скрипты разделятся на 2 части, клиентскую и серверную. В этом случае можно будет серверную не оставлять у клиентов.
