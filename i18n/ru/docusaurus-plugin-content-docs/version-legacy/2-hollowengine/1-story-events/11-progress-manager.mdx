## Меню

:::info Меню
Есть специальное меню, где будет список всего что вы добавили в менеджер ивентов. Открывается оно
на клавишу `Ё` (по умолчанию).

![ProgressManagerMenu](@site/static/img/doc/ProgressManager.png)

:::

## Управление ивентами

:::info Управление ивентами

<div align="left" id="dont-copy">

```kts
<players>.progressManager.addMessage { "<text>" } // Добавляет сообщение в менеджер ивентов
<players>.progressManager.removeMessage { "<text>" } // Удаляет сообщение из менеджера ивентов
<players>.progressManager.clear() // Очищает менеджер ивентов
```

---

```kts title="Пример"
val players by server.players

players.progressManager.addMessage { "Hello World!" }
players.progressManager.addMessage { "[Новый друг] Принеси к себе в хату нового друга." }
```
