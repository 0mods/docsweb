## Début de l’événement
:::tip sur un début
Pour démarrer un événement de scénario, tapez `/hollowengine start-script`, après que l’argument suivant indique si `ignorer les événements terminés` (true/false),
si l’événement est déjà terminé, il ne commencera pas sans `true` puis l’argument suivant trouve votre événement dans la liste.
Si vous spécifiez un nom de joueur ou un type `@a` tag après le nom, la boîte de dialogue commencera avec un joueur/groupe spécifique.
:::
