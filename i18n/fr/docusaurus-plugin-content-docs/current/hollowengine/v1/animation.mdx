# 2.2 | Dialogues | Animation
## Créer des animations de script simples
Le principe de fonctionnement de toute animation est de changer l’état de l’objet (moba / images) au fil du temps.
Par conséquent, si vous voulez faire une animation de l’augmentation du personnage, cela se fait comme suit:
```kotlin
for(i in 0..100) {
    object.scale = 1.0f + i / 100f //ajouter la taille actuelle à la taille d’origine, car si i = 50, ce sera : 1f + 0.5f, à i = 75 : 1f + 0.75f, etc.
    wait(0.1f) // attendre 1/10 secondes
}
```

C’est la façon la plus primitive d’animer n’importe quoi, et vous pouvez également [déplacer](./transform) des caractères sur l’écran de la même manière.
Par exemple, si vous déplacez tous les 10 secondes d’un caractère à une valeur aléatoire de x et y, vous pouvez animer les secousses et autres.
Il est probable qu’à l’avenir, il y aura un moyen plus simple de créer de telles animations, en interpolant sur quelques lignes :)
